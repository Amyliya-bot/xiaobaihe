# 用户使用指南

## 开始搭建

- 左侧“添加”区可直接生成方块、圆柱、球体、墙体、地面和统一人台。
- 单击物体后使用彩色控制柄；左侧可在移动、旋转和拉伸之间切换。
- 在空白画布按住左键平移视角，按住中键环绕，滚轮缩放。
- 右键拖动可框选多个物体；排列完成后单击空白处解除临时整体选择。

## 自定义模型

选择“自定义”后直接在画布落点。点之间可连线，闭合线形成面，面可沿自身法线向前或向后拉伸。进入点、线、面模式后可继续移动顶点、组织边和拉伸已有面；模型出现自交或退化风险时，软件会停在最近合法位置。

## 人台姿势

- 人台使用统一无纹理白模外观，可调整 1.2-2.2 m 参考身高，身体比例保持不变。
- 画布圆点控制的是身体段：拖动上臂点会移动肘部，拖动前臂点会移动手部；大腿和小腿同理。
- 拖动方向以当前观察视角为准。需要调整前后深度时，先把场景转到侧面再拖动对应端点。
- 肘和膝会保持在安全弯曲范围内；每次拖动松开只产生一条撤销记录。
- 站立、坐下、抬手、行走和跑步只是可继续修改的起始姿势，也可以记录到物体时间轴。

## 摄影机、灯光与动画

- 对象栏选择“主相机”会打开实时取景窗；右击“主相机”进入摄影机同步，当前画布视角会成为输出镜头，普通画布移动和对象编辑仍然可用，小窗口显示最终输出比例。
- 摄影机支持 16:9、9:16、1:1、4:3 和自定义比例。
- 时间轴在当前时刻记录摄影机或选中物体状态；播放时会在记录点之间插值。
- 用户灯光可新增面光、点光、聚光和日光，固定场景光不会被删除。

## 保存与恢复

- 第一次保存会打开 Windows 路径选择；之后“保存”更新当前工程，“另存为”重新选择位置。
- 修改后的工程会更新一份本地恢复副本，不会按时间无限堆积。
- 异常退出后再次启动会询问恢复或放弃。放弃恢复不会删除用户手动保存的工程。

## 导入与性能

可导入 GLB、GLTF、OBJ/MTL。GLTF/OBJ 依赖外部纹理或 `.bin` 时，需要在同一次文件选择中明确选中关联文件。复杂模型可能触发性能说明；选择轻量预览不会覆盖原模型，图片、视频和模型仍可单独选择原始精度输出。

## 导出

- 图片：当前画面 PNG，或同相机的白模、深度、法线、物体分色、遮罩和轮廓六通道。
- 动画：逐帧 PNG 序列或无音频 MP4/H.264。
- 模型：当前可见静态状态的 GLB、内嵌资源 GLTF 或 OBJ。

平台预设只负责校验已记录的格式、画幅、帧率、时长和大小规则，不代表平台每个入口都支持同一种参考素材。上传前请核对导出窗口中的日期和说明。

## 常见问题

- **视频无法导出**：设备可能不支持当前 WebCodecs H.264 配置，请改用逐帧 PNG。
- **导入模型不流畅**：先使用轻量预览；最终导出仍可选择原始精度。
- **模型贴图缺失**：重新导入，并同时选择 GLTF/OBJ 引用的 `.bin`、纹理或 MTL 文件。
- **工程文件很大**：当前工程会内嵌导入资产以保证可移动性，大模型会同步放大工程与恢复副本。
