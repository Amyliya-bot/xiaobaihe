# 变更记录

本项目采用语义化版本号。公开 Beta 前的里程碑仍可能调整工程格式，但读取器会保留已发布 schema 的迁移入口。

## 0.22.0 - 2026-07-19

- 面向普通用户显示的正式软件名称改为“小白盒”，同步窗口标题、编辑器页头、快捷方式、卸载项、可执行文件名和公开文档。
- 使用项目自行绘制的白底黑色粗线立方体图标，保留 SVG 母版并生成 Windows ICO、PNG 与 macOS ICNS；不引入第三方图标许可证风险。
- 保留既有应用标识、内部存储键和旧版用户数据目录，升级改名后仍可继续读取最近工程、自动恢复与本地偏好。
- 构建产物统一使用 `Xiaobaihe.exe` 和 `xiaobaihe-<版本>-setup.exe`，校验和及打包专项脚本同步更新。

## 0.21.0 - 2026-07-18

- 顶部操作按普通用户任务重组为“搭场景、调镜头、做动画”，时间轴默认收起；首次使用提供三个步骤的本地引导，帮助按钮可随时重新打开。
- 图片、视频、三维模型和高级检查统一进入“导出作品”；普通图片支持 PNG/JPG 与 1024、1280、1920、2048 四档清晰度，动画导出会同时识别人台动作。
- 本地项目中心显示最近工程数量、已知占用、文件大小和修改时间，支持打开所在文件夹、仅移除记录、清理恢复缓存，以及明确进入 Windows 回收站的删除确认。
- 人台编辑分成“整体摆放”和“调整姿势”，避免变形框与关节点同时出现；位置、旋转和大小换成大众化方向名称，高级参数继续折叠。
- 摄影机和灯光滑杆改为拖动时实时预览、松手时只记录一次撤销；场景指纹不再随播放帧重复进行整份序列化。
- 补齐对话框焦点圈定、`Esc` 关闭与焦点返回，修正导出能力判断、保存测试竞态，并增加首次引导、焦点、时间轴默认状态、滑杆撤销、JPG 与项目元数据回归。

## 0.20.4 - 2026-07-18

- 第一人称控制改为“编辑画布摄影机同步”：进入时输出摄影机无缝对齐当前画布，此后普通画布的平移、旋转和缩放实时驱动输出镜头。
- 主画布继续使用编辑相机渲染，保留无限网格、对象选择、变形控制柄、画布建模和人台关节；不再切换为隐藏辅助元素的输出画面，也不独占鼠标事件。
- 实时取景窗继续去除网格与编辑辅助元素，并独立遵守 16:9、9:16、1:1、4:3 或自定义输出比例；同步预览不会覆盖刚修改的输出画幅。
- 增加进入前后相机位置、观察中心和焦距对齐，以及模式内镜头移动、对象选中与位置编辑的组合验收。

## 0.20.3 - 2026-07-18

- 再次明确第一人称控制的显示关系：主画布切换为输出摄影机的位置、方向和焦距，拖动画布直接控制镜头。
- 主画布始终使用编辑区域自身的宽高比铺满显示，不跟随 16:9、9:16 等输出画幅裁切；独立取景窗继续严格显示最终输出比例和构图。
- 增加横向主画布与 9:16 取景窗同时渲染的桌面验收，验证镜头手势只修改输出摄影机，退出后恢复原编辑观察视角。

## 0.20.2 - 2026-07-17

- 修正第一人称控制的显示关系：主画布始终保持原编辑观察视角，鼠标手势只调整输出摄影机，真实镜头画面与比例只在实时取景浮窗中反馈。
- 摄影机在主画布中的位置框继续可见并随控制结果更新；退出第一人称控制不需要恢复主画布视角，因为编辑观察相机从未被替换。
- 保留 `0.20.1` 的真实人台动作、横竖画幅自适应和四边四角等比缩放。

## 0.20.1 - 2026-07-17

- 五个人台动作改为从原始骨骼动画按时间轴确定性采样，支持播放、暂停、手动关节覆盖，并进入逐帧图片和视频输出；不再只是固定姿势截图。
- 人台操作提示移除“官方”措辞，动作按钮显示当前选择，并在人台属性区提供直接播放/暂停入口。
- 第一人称控制期间，主画布完整切换为输出摄影机视角；退出后恢复原编辑观察相机，实时取景浮窗保持同步。
- 实时取景内容严格保持摄影机画幅比例；切换横竖画幅时近似保持浮窗面积，并支持从四条边和四个角等比缩放。
- 增加取景窗比例与八方向缩放单元测试，以及真实骨骼变化、主镜头视角和横竖画幅 Electron 验收。

## 0.20.0 - 2026-07-17

- 人台改用 Quaternius `Universal Animation Library` Standard 免费包中完全匹配的官方人台与骨架；五个基础姿势从官方动作片段采样，继续支持画布关节微调、保存、撤销、时间轴和模型导出。
- 实时取景窗改为可在主画布内自由拖动和缩放的独立浮窗，输出画幅变化时保持真实比例并限制在可操作区域内。
- 右击对象栏“主相机”可进入“第一人称控制”：左键平移、中键转向、滚轮前后移动，五秒操作提示自动消失，`Esc` 退出但保留取景窗。
- 修正时间轴高度少一像素造成的最后一行裁切；在 1280×720 下验证轨道、记录点和总时长控制完整可见。
- 工程 schema 升级到 12，保存官方姿势标识和手动关节覆盖；schema 1-11 均保留迁移入口。

## 0.19.0 - 2026-07-16

- 使用经许可证审计的 Quaternius CC0 人形骨架模型替换胶囊式人台外观，保持统一白模材质、固定比例和既有工程姿势数据。
- 四肢控制改为拖动肘、手、膝、脚等实际端点；鼠标目标随当前相机方向计算，关节长度固定并继续应用安全角度限制。
- GLB 导出剔除运行时控制数据并保留骨骼；OBJ 会把当前姿势烘焙为静态顶点，不回到绑定姿势。
- 新人台已验证姿势预设、时间轴、工程保存、GLB/OBJ 导出以及 1440x900/1280x720 黑白主题。

## 0.18.0 - 2026-07-16

- 面向公开 Beta 的 Windows 发行候选。
- 移除窗口“开发版”标识，补齐安装包元数据、许可、隐私、用户指南和校验流程。
- 生成运行时第三方许可证文本与完整锁定依赖许可证清单。

## 0.17.0 - 2026-07-16

- 增加复杂导入模型风险提示、可逆轻量预览和独立最终导出精度。
- 工程 schema 升级到 11，并迁移 schema 1-10。

## 0.16.0 - 2026-07-16

- 增加确定性 MP4/H.264 导出、配置化平台预设和安全取消。

更早里程碑属于公开 Beta 前的内部开发记录，暂不作为公开变更日志发布。
